== Description ==

Pulls your configurable list of Twitch channels from [scripts.platypusmuerte.com](https://scripts.platypusmuerte.com) and displays them, grouped by online then offline channels. The outputs CSS can easily be overridden and adjusted for your theme by using WordPress additional CSS menu.
 
## Examples
**[PlatypusMuerte.com/community/streams](https://platypusmuerte.com/community/streams/)** 
**[PTAgamers.com/streamers/](https://ptagamers.com/streamers/)**

== Installation ==

Download and install the plugin. The plugin settings page has a field to enter your key from [scripts.platypusmuerte.com](https://scripts.platypusmuerte.com). A Patreon account is required for login only.
